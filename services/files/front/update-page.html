Gestion des données de l'ontologie

Cette section vous permet de gérer les analyses empiriques stockées dans l'ontologie IA-DAS. Vous pouvez ajouter de nouvelles analyses ou modifier celles existantes.

Ajouter une analyse : Créez une nouvelle entrée avec toutes les informations requises (article, population, variables, relations statistiques, etc.).

Modifier une analyse : Sélectionnez une analyse existante par son ID pour la consulter et la modifier.

⚠️ Important

Les modifications apportées seront directement sauvegardées dans la base de données Fuseki. Assurez-vous de la validité des informations avant de confirmer.
